**Agency Website Template**
**Overview**
This is a responsive and dynamic website template designed for digital agencies. Built using HTML5, CSS3, and Bootstrap 5, it includes multiple sections like a navigation bar, services, testimonials, and social media integration. It features a clean and modern design with smooth scrolling for enhanced user experience.

**Features**
Responsive Navigation Bar:
Includes links to key sections like Home, Services, About Us, Testimonials, and Contact.

Banner Section:
A hero section with a call-to-action (CTA) button and an image, designed to grab user attention.

Services Section:
Displays the core services offered, each with an image, title, and description.

About Us Section:
Highlights the unique selling points of the agency, including a bulleted list of key reasons to choose the services.

Testimonials Section:
Features client reviews with images and details to build trust.

Social Media Section:
Links to the agency's social media profiles with recognizable icons.

Footer Section:
Includes contact details, a subscription form, and copyright information.

Smooth Scrolling:
Enables smooth transitions when navigating to different sections.

**Tech Stack**
HTML5: For structure and content.
CSS3: For styling and layout.
Bootstrap 5: For responsive design and pre-built components.
JavaScript: For smooth scrolling functionality.
